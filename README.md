# Polyhedral Complex Derivation from Piecewise Trilinear Networks

This is the repository that contains source code for the [TropicalNeRF website](https://tropical-nerf.github.io).


## Build and deploy

```bash
npm run build  # build 
npm run deploy # deploy
```

## Local demo

```bash
npx vite
```
